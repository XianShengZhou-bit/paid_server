# Frontend 编译与运行

Frontend 为静态页面 + Nginx 反代，无需编译。

## 前置条件

安装 Nginx：

```bash
sudo apt install nginx
```

项目根目录需有 `.env`（可从 `.env.example` 复制）。

## 一键运行

在项目根目录执行：

```bash
./frontend/start_nginx.sh
```

默认访问地址：`http://127.0.0.1:9994`（由 `PAYMENT_NGINX_IP`、`PAYMENT_NGINX_PORT` 决定）。

- 登录页：`/login`
- API 反代：`/api/*` → Gateway HTTP
- WS 反代：`/ws/*` → Gateway WebSocket

## 日志

`start_nginx.sh` 前台运行时，访问日志与错误日志输出到终端（stdout / stderr）。

生成的 Nginx 配置：`.run/nginx/nginx.conf`

## 停止

前台运行时按 `Ctrl+C`。
