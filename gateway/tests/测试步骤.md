# Gateway 单元测试

## 编译（首次或改 CMake 后）

在项目根目录执行：

```bash
cmake -S gateway/tests -B gateway/tests/build \
  -DPAYMENT_TEST_ENV_FILE="$(pwd)/.env.test"
cmake --build gateway/tests/build -j"$(nproc)"
```

## 一键运行全部测试

```bash
ctest --test-dir gateway/tests/build --output-on-failure
```

## 手跑（可选）

```bash
./gateway/tests/build/gateway_tests .env.test
```
