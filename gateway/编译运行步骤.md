# Gateway 编译与运行

## 编译（首次或改 CMake 后）

在项目根目录执行：

```bash
cmake -S gateway -B gateway/build
cmake --build gateway/build -j"$(nproc)"
```

## 一键运行

```bash
./gateway/build/payment_gateway .env
```

`.env` 为环境变量文件路径，也可换成 `.env.test` 等。

启动后会同时监听 HTTP（`PAYMENT_GATEWAY_HTTP_PORT`）与 WebSocket（`PAYMENT_GATEWAY_WEBSOCKET_PORT`）。

## 日志

日志路径由 env 里的 `LOG_FILE` 决定（见 `.env.example`）：

| 配置项 | 说明 |
|--------|------|
| `LOG_FILE` | 日志文件路径，默认示例：`/tmp/payment_backend.log` |
| `LOG_CONSOLE_ENABLED` | 是否同时输出到终端 |

启动成功后会打印：`日志系统初始化完成: file=...`

## 停止

前台运行时按 `Ctrl+C`，或：`kill -TERM <pid>`
